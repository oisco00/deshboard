주간업무 · 수시긴급 공유앱
==============================

1. 실행 방법
- 압축을 해제합니다.
- index.html 파일을 더블클릭하거나 휴대폰 브라우저로 열면 됩니다.
- 인터넷 연결 없이도 동작합니다.

2. 기본 로그인
- 관리자: admin / admin123!
- 시연용: demo / demo1234

3. 주요 기능
- 첨부 엑셀 구조를 반영한 업무 등록
- 주간보고 / 수시긴급 분리 입력
- 이미지 / PDF 첨부 저장
- 등록자 추적
- 본인 및 관리자만 수정/삭제 가능
- 일자별 조회 / 검색 / 필터
- 주간 종합보고 생성
- HTML 다운로드
- PNG 다운로드
- 브라우저 인쇄를 통한 PDF 저장
- JSON 백업 / 복원

4. 현재 버전의 특징
- 브라우저 내부 저장소(IndexedDB)에 저장되는 로컬 실행형 앱입니다.
- 같은 기기에서는 계속 데이터가 유지됩니다.
- 다른 휴대폰/PC와 실시간으로 공유되는 서버형 시스템은 아닙니다.
- 여러 사람이 동시에 공동 사용하려면 Firebase / Supabase / 사내 서버 DB 방식으로 확장해야 합니다.

5. 첨부 엑셀 반영 내용
- 업로드 파일: 주간업무 회의록 250409.xlsx
- 날짜형 시트 데이터를 초기 데이터로 불러와 두었습니다.
- 초기 데이터 건수: 932건

6. 권장 사용 방식
- 휴대폰에서는 브라우저에서 연 뒤 "홈 화면에 추가"로 앱처럼 사용
- 주기적으로 JSON 백업 파일 저장
- 최종 보고 공유 시 HTML 다운로드 또는 PDF 저장 사용

7. 다음 단계 확장 가능
- 회사 공용 로그인
- 관리자 승인 프로세스
- 서버 DB 연동
- 카카오톡 공유 버튼 연동
- 엑셀 자동 업로드 / 재생성



8. 프롬프트
첨부 화일은 주간보고 및 수시긴급 사항을 카카오톡 단체방으로 제시된 내용을 정리하여 주간의 내용을 대표자 및 간부들이 정보를공휴하고 있는 행위를 프로그램을 만들어 핸드폰을 이용하여 카카오톡에 입력하던 내용을 입력하게 하고 첨부화일 이미지를 등록하고 수정 할 수 있도록 하고
등록된 자료는 보관이 되어 언제던지 볼수 있도록 하고
사용자 로그인으로 누가 올린자료인지를 알수 있게 하고
본인과 관리자만이 자료를 수정 할 수 있도록 함.
업무내용 등록시에는 현재의 첨부화일을 분석하여 입력 형태 밀 길이등을 반영하고
첨부 화일 또한 등록되어 일자별 개별 등록 내용 확인 및 수정, 전체 합계된 보고용 보고서를 확인할 수 있도록 하고
이미지나 pdf로 다운로드 공유 할 수 있도록 해야함
